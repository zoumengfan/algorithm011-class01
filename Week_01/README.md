## Array
- Lookup: O(1)
- Insertion & Deletion: O(n)

## LinkedList
- Lookup: O(n)
- Insertion & Deletion: O(1)

## Skip List
- Only for sorted elements
- Compare to AVL Tree & Binary Search
- Insertion/Deletion/Lookup: O(logn)
- Use Case: Redis

### Queue, Priority Queue
